Student Project Management System

A full-stack web application built using ReactJS and Spring Boot that allows students to submit project details and manage them through a dashboard. The system provides features like searching, department filtering, and exporting project data for better organization.

Features

Student project submission form

Department selection using dropdown

Dashboard to view submitted projects

Search functionality

Department filtering

Export project details to Excel

Export project details to PDF

Clean and professional user interface
